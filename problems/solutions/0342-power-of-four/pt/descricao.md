# 342. Power of Four (Fácil)

Dado um número inteiro `n`, retorne _ `TUE` se for um poder de quatro.De outra forma,
retornar `false`_.

Um número inteiro `n` é um poder de quatro, se existe um número inteiro` x` tal que `n
== 4x`.

## Exemplo 1:



Entrada: n = 16
Saída: true


## Exemplo 2:



Entrada: n = 5
Saída: false


## Exemplo 3:



Entrada: n = 1
Saída: true


## restrições

* `-231 <= n <= 231 - 1`
